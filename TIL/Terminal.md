# 터미널 기본 명령어



| **Is**                                 | list 조회하기                         |
| -------------------------------------- | ------------------------------------- |
| **cd 디렉토리명**                      | change directory 해당 디렉토리로 이동 |
| **cd ..**                              | 상위 디렉토리로 이동                  |
| **cd 또는 cd~**                        | 홈 디렉토리로 이동                    |
| **rmdir 디렉토리명**                   | remove directory 폴더 삭제            |
| **touch 파일명.확장자명**              | 파일 생성                             |
| **clear 또는 cmd+k**                   | 화면 클리어하기                       |
| **cp 대상파일명 변경파일명 타겟 경로** | copy 복사                             |
| **mv**                                 | move 이동                             |
| **rm**                                 | 삭제                                  |

| whoami          | 내 정보 조회         |
| --------------- | -------------------- |
| **cat**         | 텍스트 파일 열기     |
| **vim 또는 vi** | vim 편집기 실행      |
| **grep **       | 파일 안 내용 찾기    |
| **sudo**        | 관리자 권한으로 실행 |
